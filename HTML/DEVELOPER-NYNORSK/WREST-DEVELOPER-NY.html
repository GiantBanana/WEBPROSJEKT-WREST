WRest Dokumentasjon

WRest(Wcag request standard) er en tilretteleggingsprotokol for å bruke med WCag standarden eller alene for å gi bruker mulighet til å tilpasset nettstedet sine behov. Ved å gjøre brukerns infromasjon tilgjengelig både på tjenersiden og clientsiden håper vi å tilby en fleksibel løsning for utviklere og brukere, da de slipper å spesifisere tilpassingen for hver nettside.

Netttjener

wRest gir informasjon i "JSON" formattet og ser slik ut.

{ "highcontrast":"on", "plainText":"on", "screenReader":"off" }

Forespørselen er sendt via "HTTP Headers", ved å bruke PHP eller annen form for standard tjenerside prossessering, er det enkelt å hente ut informasjonen brukere har sendt.

Denne koden vil la PHP identifisere og prossesere wRest informasjonen og lagre den i variablen $wRestConfig, og om vi da kjører koden var_dump($wRestConfig); får vi følgende utskrift.

array(3) { ["highcontrast"]=> string(2) "on" ["plainText"]=> string(2) "on" ["screenReader"]=> string(3) "off" }

Nå kan vi se hva brukerns behov er og gjøre handligner for å gjøre opplevelsen bedre for brukern. Og mindre hodebry for Utvikler.

Eksempel Php

I dette eksempelet viser vi hvordan du kan laste to forskjellige CSS filer bassert på om bruker har etterspurt høy kontrast på siden din.

<head> <?php $wRestConfig; // Getting http wRest header foreach(getallheaders() as $name => $value){ if($name == "Wrest"){ $wRestConfig = json_decode($value, true); } } // Checking if user requested highcontrast if($wRestConfig['highconstrast'] == "on"){ ?> <link href="highconstrast.css" rel="stylesheet" type="text/css"> <?php }else{ ?> <link href="normal.css" rel="stylesheet" type="text/css"> <?php } ?> </head>

Nettleser

Ved å legge til en div knagg med navnet "wRESTConf", kan nettleseren vite hvor den skal plassere wRest informasjonen (JSon objekt), slik at javascript kan få tilgang og gjøre tilpassninger.

<div hidden id="wRESTConf"> <!-- Here the JSON information from WRest will appear --> </div>

Les fra div knagg

var wrestConfig; document.addEventListener("DOMContentLoaded", function(event) { wrestConfig = JSON.parse(document.getElementById("wRestConf").innerHTML); });

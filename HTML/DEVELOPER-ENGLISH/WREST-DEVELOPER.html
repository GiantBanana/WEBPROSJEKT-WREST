WRest Documentation

WRest(Wcag request standard) is an accessebility protocol for use with the Wcag standard or as a stand alone customization alternative. By serving this information booth on the server side and client side it's easy to costomize your webpage for the user without having the user request it per site.

Server Side

The request is sendt with the HTTP Headers, so by using standard php or other serverside language you can easily get the infromation with the request

<?php echo'<?php //Get the WRest Json config file foreach(getallheaders() as $name => $value){ if($name == "Wrest"){ echo "<h1> your Wrest code is " . $value . " !!! wop wop </h1>"; } } ?>';?>

Runtime

By implementing the div tag with name "wRESTConf" will the browser know where to put the json config object, and you can then use javascript to access it during runtime

<!-- Here the JSON information from WRest will appear --> <div hidden id="wRESTConf"> </div>

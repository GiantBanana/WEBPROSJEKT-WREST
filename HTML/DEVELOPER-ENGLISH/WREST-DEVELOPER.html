WRest Documentation

Arrow from Browser icons to server icon showing sending of information

WRest(Wcag request standard) is an accessebility protocol for use with the Wcag standard or as a stand alone customization alternative. By serving this information both on the server side and client side it's easy to costomize your webpage for the user without having the user request it per site.

Server Side

The json input from wRest is as follows

{ "highcontrast":"on", "plainText":"on", "screenReader":"off" }

The request is sendt with the HTTP Headers, so by using standard php or other serverside language you can easily get the infromation with the request

Running this code will allow PHP to find and process the wRest tag in the HTTP header, and its now stored in the $wRestConfig variable, and if we do a var_dump($wRestConfig); we get this output.

array(3) { ["highcontrast"]=> string(2) "on" ["plainText"]=> string(2) "on" ["screenReader"]=> string(3) "off" }

At this point you can check the users needs and do approriate actions to accommodate the users need, making it smoother for booth user and developer.

Example Php

In this example we show how you can load a seprate CSS for users requesting high constrast pages.

<head> <?php $wRestConfig; // Getting http wRest header foreach(getallheaders() as $name => $value){ if($name == "Wrest"){ $wRestConfig = json_decode($value, true); } } // Checking if user requested highcontrast if($wRestConfig['highconstrast'] == "on"){ ?> <link href="highconstrast.css" rel="stylesheet" type="text/css"> <?php }else{ ?> <link href="normal.css" rel="stylesheet" type="text/css"> <?php } ?> </head>

Runtime

By implementing the div tag with name "wRESTConf" will the browser know where to put the json config object, and you can then use javascript to access it during runtime

<div hidden id="wRESTConf"> <!-- Here the JSON information from WRest will appear --> </div>

Read from the Wrest div tag

var wrestConfig; document.addEventListener("DOMContentLoaded", function(event) { wrestConfig = JSON.parse(document.getElementById("wRestConf").innerHTML); });

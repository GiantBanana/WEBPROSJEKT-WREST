Wrest

What is Wrest?

WCag Request Standard (Wrest from now on) is a standard created by five students at the Oslo and Akershus University College. Our vision is to make it a bit simpler for developers to adjust content their content in order to make it more accesible for people with different kinds of disabilities. To acomplish this we have created a simplified version of the Web Content Accessibility Guidelines 2.0 (WCAG from now on) so that it will be easyier to understand. In addition we have created a web browser function that addapts the web content to the individual user.

Why Wrest?

The Norwegian government has created legislation banning disctimination based on disabilities and making it manditoyry to design according to universal desgin standards. The law states that, from Januray 2021, every ICT system should be universally designed.
